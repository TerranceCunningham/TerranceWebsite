INFORMATION ARCHITECTURE

Project: Home Performance Washington Information Architecture.

Home Performance Washington's Information Architecture. was designed to have a main page and an inner page for our client. It was also discussed that the client wanted a contact page and callender section to keep track of lastest posts from clients and members, Home Performance Washington would then be integrated to Wordpress for the client.

WIREFRAMING

Project: Home Performance Washington Wireframe and Layout

As Home Performance Washington is a group project it was decided that three different wireframes would be shown to our client. My Wireframe was chosen to be built due to cleanliness and easy navigation.

DESIGN SPECIFICATION

Project: Home Performance Washington Design Specification used to show client measurements.

Before Home Performance Washington was to be handed off we created a design specification so that our client would be able to see the measurements, colors and images we used for each section. HPW Design Specification will help our client know the exact measurements so if they ever run into a problem later on they will have the correct base structure measurements to go off from.
